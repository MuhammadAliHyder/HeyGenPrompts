# HeyGen Generations

This folder contains a collection of prompts designed for generating videos using HeyGen, an AI-powered video creation platform. These prompts are tailored for various use cases, including consistency testing, marketing advertisements, and customizable templates.

## Files Overview

- **consistencyPrompt**: A prompt for creating a 5-second avatar video focused on consistency testing. It ensures a single static scene with a neutral avatar delivering a specific script about Google Workspace.

- **consistencyPrompt2**: Another version of the consistency prompt (content not detailed here, but likely similar to the first for testing variations).

- **consistencyPrompt3(Scenery)**: A 10-second avatar-less video prompt for output consistency testing. It describes a calm urban street scene with nature elements, emphasizing visual similarity across generations.

- **googleAvatarlessPrompt**: A 30-second avatar-less marketing advertisement for Google. This prompt includes a full script, scene structure, and specifications for motion graphics, B-roll, typography, and logos to promote Google Workspace.

- **inovioAvatarlessPrompt**: A 30-second avatar-less marketing advertisement for Inovio Tech. Similar to the Google prompt, it features a script, scene breakdowns, and visual elements to highlight Inovio's technology solutions and partnerships.

- **variblePromptGuide**: A customizable template for creating avatar-driven marketing videos. It uses placeholders (e.g., [VIDEO_LENGTH], [COMPANY_NAME]) to allow for variable inputs, making it adaptable for different brands or scenarios.

## Usage

To use these prompts:
1. Open HeyGen and select the appropriate video generation mode (avatar or avatar-less).
2. Copy and paste the content of the desired prompt file into the prompt input.
3. Customize any variable placeholders if using the guide.
4. Generate the video and review for consistency or marketing effectiveness.

These prompts are primarily for testing AI video generation consistency and creating professional marketing content. Ensure compliance with HeyGen's terms of service and any brand guidelines when using logos or specific branding.

## Notes

- All prompts emphasize specific constraints to maintain output consistency.
- Avatar prompts include detailed settings for avatar behavior, camera, and voice.
- Avatar-less prompts focus on visuals, scripts, and motion graphics.
